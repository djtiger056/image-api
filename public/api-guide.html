外部调用说明

本服务对外提供 OpenAI 风格的图片接口，并同时兼容 Jimeng / Kling / Doubao (豆包) 三条链路。

如果你在服务端保存了凭证，调用时通常只需要传模型和提示词；如果部署了 x-api-key，再额外带上请求头即可。

1. 健康检查

curl http://127.0.0.1:18080/ping

2. 模型列表

curl http://127.0.0.1:18080/v1/models

3. Jimeng 生图

curl -X POST http://127.0.0.1:18080/v1/images/generations \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "jimeng-4.5",
    "prompt": "一朵白色小花，极简插画风，纯净背景",
    "ratio": "1:1",
    "resolution": "1k"
  }'

4. Kling 网页模式生图

curl -X POST http://127.0.0.1:18080/v1/images/generations \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "kling-v2-1",
    "prompt": "一只在霓虹雨夜里的小狐狸，电影感，高清插画",
    "async": true,
    "provider_options": {
      "transport": "web",
      "target_url": "https://klingai.com/app/image/new"
    }
  }'

5. Doubao (豆包) 生图

curl -X POST http://127.0.0.1:18080/v1/images/generations \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "doubao-seedream-4.5",
    "prompt": "一只在星空下的白色猫咪，梦幻水彩风格",
    "ratio": "1:1",
    "resolution": "1k"
  }'

5.1 Doubao (豆包) 视频生成

curl -X POST http://127.0.0.1:18080/v1/doubao/videos/generations \
  -H 'Content-Type: application/json' \
  -d '{
    "model": "doubao-seedance-2.0-fast",
    "prompt": "一只小猫在草地上欢快地奔跑",
    "ratio": "16:9"
  }'
  • 模型：doubao-seedance-2.0-fast（Seedance 2.0 Fast，每日10次免费额度）
  • 支持比例：16:9 9:16 1:1 4:3 3:4
  • 每次生成5秒视频，消耗1额度
  • 返回格式与图片接口一致：{"data": [{"url": "视频URL"}]}
  • 额度用完时返回：{"error": {"type": "quota_exhausted"}}

6. 带 API Key

curl -X POST http://127.0.0.1:18080/v1/images/generations \
  -H 'Content-Type: application/json' \
  -H 'x-api-key: your-api-key' \
  -d '{"model":"jimeng-4.5","prompt":"hello"}'

7. 说明

  • Jimeng 凭证可通过控制台保存为 sessionid 或 Authorization。
  • Kling 凭证可直接粘贴 storageState JSON 或 cookies 导出 JSON。
  • Doubao 凭证通过环境变量 DOUBAO_AUTHORIZATION 或 DOUBAO_SESSIONID 配置，也可在请求头中直接传入。
  • 如果服务配置了 API Key，控制台和外部调用都要带上 x-api-key。
